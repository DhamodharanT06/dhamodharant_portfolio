# Dhamodharan Portfolio

Local dev:

1. Install dependencies
2. Start dev server

## Scripts
- npm run dev – start Vite dev server
- npm run build – production build
- npm run preview – preview build

## Troubleshooting
- If Tailwind classes don’t apply, ensure tailwind.config.js content globs include ./index.html and ./src/**/*.{js,jsx,ts,tsx}.
- If OneDrive blocks file watching, try moving the project to a non-synced folder or start Vite with --host.
